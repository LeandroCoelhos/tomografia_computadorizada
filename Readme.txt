# Modelagem Matemática de um problema de tomografia computadorizada

Esta pasta tem a finalidade de centralizar os estudos da matéria, e tem finalidade totalmente acadêmica. 
A implementação não é completa ou modela o mundo real em sua totalidade, mas é uma boa forma de entender como imagens são geradas a partir de máquinas como as de tomografia.

Este projeto tem como objetivo apresentar um modelo de reconstrução de imagem a partir do escaneamento por raios X de uma seção transversal do corpo humano utilizado no exame conhecido por Tomografia Computadorizada.
Para construir cada imagem, são emitidos feixes de Raios X que ultrapassam o corpo do paciente e são captados por detectores. A fração de fótons da radiação que não é absorvida ou desviada pelo corpo é captada por detectores que convertem a energia incidente em corrente elétrica, proporcional à potência dos Raios X originais, que por sua vez é convertida em sinais eletrônicos que são enviados a um computador, que constrói as imagens.
